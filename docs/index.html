features

  • ▸
    MCP Server38 tools — just say "play some lofi" or "skip this" inside Claude Code.
  • ▸
    /cwb SkillRouting brain: mood → smart search, artist → catalog, "play 3" → play_number. No guessing.
  • ▸
    Apple Music + Local + QQAppleScript-driven, no GUI needed. Local afplay fallback. QQ Music 30s preview.
  • ▸
    PlaylistsList and play Apple Music playlists by name — user-created or subscription. Queue syncs to cwb watch.
  • ▸
    Play HistoryRecently played tracks from Apple Music's native history. history_search recommends new music based on your listening patterns.
  • ▸
    Vibe EngineHooks track git commits, test runs, and tool calls — buddy reacts in real time.
  • ▸
    Karaoke & Live PlayerFull-screen LRC lyrics, album art in ANSI half-block, GameBoy retro border.
  • ▸
    StatuslineOne line: mood face + track + progress bar + beat wave + current lyric.
