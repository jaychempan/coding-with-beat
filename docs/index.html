features

  • ▸
    MCP Server 21 tools — just say "play some lofi" or "skip this" inside Claude Code.
  • ▸
    Apple Music + Local + QQ AppleScript-driven, no GUI needed. Local afplay fallback. QQ Music 30s preview.
  • ▸
    Vibe Engine Hooks track git commits, test runs, and tool calls — buddy reacts in real time.
  • ▸
    Karaoke & Live Player Full-screen LRC lyrics, album art in ANSI half-block, GameBoy retro border.
  • ▸
    Statusline One line: mood face + track + progress bar + beat wave + current lyric.
